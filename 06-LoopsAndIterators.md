# Looping and Iterators

## 1.) While Loops
```python
counter = 0

while counter < 10:
    print("Hello World!")
    counter = counter + 1

    #Result:  #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
              #Hello World!
```
## 2.) For Loops
```python
list_of_accessories = ['Scarf', 'Socks', 'Shoes', 'Belt']

for clothes in list_of_accessories:
     print(clothes)

#Result:  #Scarf
          #Socks
          #Shoes
          #Belt 

```
